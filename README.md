scalicifarm.co
======
**scalicifarm.co** is a contract site built for Scalici Farm & Greenhouses. The goal of the site is to inform customers of business hours, location, services provided and merchandise available for purchase.

## Technology
* HTML5
* CSS3
* Jquery
